## Pix-System

This is a frontend application that simulates the Pix system, developed as part of a college project.

---

## Requirements
To access all the features, make sure the backend service **pix-system-back** is running.

---

## Getting Started

Run the command below to download the dependencies.

```bash
npm install
# or
yarn
```


Run the development server: 

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
